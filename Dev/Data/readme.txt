In Diesem Ordner werden Powershell-Skript-Dateien abgelegt die lediglich Variablen enthalten.

Die Namensgebung kann recht frei gewählt werden.
Präfix: PS-Data_xxxx.ps1

-) Die Bezeichnung der Datei sollte eine Beziehung zu der verwendenden Funktion erkennen lassen.
-) Eine Datei kann durchaus mehrere Variablen enthalten
-) Die Variablen sollten so "abstrakt" gewählt werden um keine Konflikte zu erzeugen

Diese Dateien müssen auch signiert sein
